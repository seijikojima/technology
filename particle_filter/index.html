Monte Carlo Localization using Particle Filter

Monte Carlo Localization 2D map example \( (x,y,\theta) \) \( X_t \) depends only on \( X_{t-1} \)

equation of state model

1. move and predict with particles

\[ P( \boldsymbol{x} | \boldsymbol{u}) = N () \]

2. update particles by sensor value

\[ P( \boldsymbol{x} | \boldsymbol{z}) \]

3. resampling

pseudocode


    _Xt = 0                                    
    Xt = 0                                     
    for m in range(0,M):                       
        xt[m] = motion_update(ut,xt-1[m])      
        wt[m] = sensor_update(zt,xt[m])        
        _Xt = _Xt + (xt[m],wt[m])              
                                               
    for m in range(0,M):                       
        draw xt[m] from _Xt with prob wt[m]    
        Xt = Xt + xt[m]                        
                                               
    return Xt
