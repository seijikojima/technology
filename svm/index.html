Boundary function

\begin{align} f(x) = \boldsymbol{w^T x} + \boldsymbol{b} = 0 \end{align}

Each \(x_i\) is in eather \(C_1 \) or \( C_2 \)

\begin{align} x_i \in C_1 \Leftrightarrow \boldsymbol{w^T x} + \mathbf{b} < 0 \end{align} \begin{align} x_i \in C_2 \Leftrightarrow \boldsymbol{w^T x} + \mathbf{b} > 0 \end{align} using label \(t_i\) and discribe above two equations \begin{eqnarray} t_{i}= \left\{ \begin{array}{ll} -1 & ( x_i \in C_1 ) \\ 1 & (x_i \in C_2 ) , \ \ \ (i = 1,2,...,n) \\ \end{array} \right. \end{eqnarray} \[ \]

The distance \( d \) between datapoint \(\boldsymbol{x_i} = (x_1,x_2,...,x_p) \) and f(x). This dimension of \(x_i\) is \(p\)

\[ d = \frac{ \| \boldsymbol{w^Tx} + \boldsymbol{b} \|}{ \| \boldsymbol{w} \|} \]

maximize margin \(M\)

\[ \arg \max_{\boldsymbol{w,b}} M, \ \ \ \frac{ \boldsymbol{t_i}( \boldsymbol{w^Tx_i} + \boldsymbol{b} )}{ \| \boldsymbol{w} \|} > M, \ \ \ (i=1,2,...,n) \]
